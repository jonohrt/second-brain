1. **ios/SecondBrain/Services/AudioRecorder.swift**:
   - Class `AudioRecorder` (not actor -- needs synchronous start/stop from UI thread)
   - Private properties: `engine: AVAudioEngine`, `audioFile: AVAudioFile?`, `recordingURL: URL?`
   - `func startRecording() throws -> URL`:
     - Configure AVAudioSession: category `.playAndRecord`, options `.defaultToSpeaker`, setActive true
     - Create temp file URL: `FileManager.default.temporaryDirectory.appendingPathComponent(UUID().uuidString + ".wav")`
     - Get input node format: `engine.inputNode.outputFormat(forBus: 0)`
     - Create AVAudioFile for writing at temp URL with the format settings
     - Install tap on input node (bufferSize: 1024) that writes buffers to file
     - engine.prepare() then engine.start()
     - Store and return the URL
   - `func stopRecording() -> URL?`:
     - Remove tap from input node
     - Stop engine
     - Set audioFile to nil
     - Return the stored recordingURL

2. **ios/SecondBrain/Services/TranscriptionService.swift**:
   - Class `TranscriptionService`
   - Private property: `whisperKit: WhisperKit?`
   - `var isReady: Bool` computed property (whisperKit != nil)
   - `func initialize() async throws`:
     - Create WhisperKitConfig with model: "base" (per research -- English-focused, ~142MB)
     - Initialize WhisperKit with that config
     - Store in whisperKit property
   - `func transcribe(audioURL: URL) async throws -> String`:
     - Guard whisperKit is initialized, throw TranscriptionError.notInitialized otherwise
     - Call `whisperKit!.transcribe(audioPath: audioURL.path)`
     - Return the text result, or empty string if nil
   - `enum TranscriptionError: Error, LocalizedError` with cases: `notInitialized`, `transcriptionFailed(String)`
     - Provide errorDescription for each

3. **ios/SecondBrainTests/TranscriptionServiceTests.swift**:
   - Test that TranscriptionService starts with isReady == false
   - Test that transcribe() before initialize() throws notInitialized
   - Note: Full WhisperKit integration requires device/simulator with model download; these tests cover the guard logic only

1. **ios/SecondBrain/ViewModels/AppViewModel.swift**:
   - `@Observable @MainActor class AppViewModel`
   - Published state properties:
     - `transcription: String = ""` (editable text field binding)
     - `answer: String = ""` (LLM response)
     - `isRecording: Bool = false`
     - `isTranscribing: Bool = false` (shows "Transcribing..." indicator)
     - `isLoading: Bool = false` (shows loading indicator during API call, VOICE-03)
     - `error: String? = nil` (error message for RESP-02)
     - `isWhisperReady: Bool = false` (tracks model download state)
     - `setupMessage: String? = "Downloading speech model..."` (first-launch indicator per research pitfall 1)
   - Private services: `apiClient: APIClient`, `recorder: AudioRecorder`, `transcriber: TranscriptionService`
   - `init()`: create all three services with defaults
   - `func initializeWhisper() async`:
     - Set setupMessage to "Downloading speech model..."
     - try await transcriber.initialize()
     - Set isWhisperReady = true, setupMessage = nil
     - On error: set setupMessage to "Failed to load speech model: \(error.localizedDescription)"
   - `func startRecording()`:
     - Guard isWhisperReady, else return
     - Clear previous answer and error
     - Set isRecording = true
     - try recorder.startRecording() -- catch and set error
   - `func stopRecording()`:
     - Set isRecording = false
     - Guard let url = recorder.stopRecording() else return
     - Set isTranscribing = true
     - Task: try await transcriber.transcribe(audioURL: url), set transcription, catch and set error
     - Set isTranscribing = false
   - `func sendQuestion() async`:
     - Guard transcription is not empty (trimmed)
     - Set isLoading = true, error = nil
     - do: let response = try await apiClient.ask(text: transcription)
     - Set answer = response.answer
     - catch: set error = error.localizedDescription
     - Set isLoading = false
   - `func retry() async`: just call await sendQuestion()
